1. SDK API

Added

xxxxxxxxxx
 boolean provideForceProvisionRequest();
  • Try when all DRM content suddenly fails to play on the device and there is an error like the one below. Failed to restore keys: General DRM error (-2000)
  • This error can occur when there is a problem with provisioning and can be reset with the provideForceProvisionRequest() function.
  • You must be connected to the network when using the function.

Importent :

  • Frequent use of this function may cause other errors.
  • If there is still a problem after using this function, you may need to reinstall the app or factory reset the device.
  • Saved Offline licenses are removed when the provision is reset. This is because an error occurs when using a previous offline license.
